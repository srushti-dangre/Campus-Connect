# CampusConnect

CampusConnect is a **college event management web app** that allows students and administrators to stay updated with all campus events. Built using **HTML, CSS, and JavaScript**, this app is lightweight, responsive, and easy to run.

---

## Project Overview

- **Students** can view events, set reminders, and toggle dark mode.  
- **Admins** can add, edit, and delete events.  
- Events and settings are saved in **localStorage**.  
- Modern **glassmorphism UI** and fully responsive design.  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript  

---

## How to Run

1. Clone or download the repository.  
2. Open the folder in **VS Code** or any code editor.  
3. Open `index.html` using **Live Server**.  
4. Login with college, student name, admission number, and password.  
   - Password with `@admin` gives **admin access**.  
5. Use the dashboard to view events, add/edit/delete events (admins), or set reminders (students).
